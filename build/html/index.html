Je ne suis en aucun cas lié aux éditeurs des logiciels

Canopée héberge de la documentation sur les logiciels de DAO les plus utiles, de l’esquisse au BIM. L’ensemble du site est compilé en html avec Sphinx à partir de documents sources écrits avec reStructuredText and Markdown. Les pages html du site sont hébergées sur Github en utilisant Git.
